# My Portfolio Website

This is my personal portfolio website showcasing my projects and skills.

## Structure

- `index.html` - Main HTML file
- `css/style.css` - Stylesheet
- `js/main.js` - JavaScript functionality
- `images/` - Directory for images

## Setup

1. Clone this repository
2. Customize the content in `index.html`
3. Add your own styles in `css/style.css`
4. Add any additional JavaScript functionality in `js/main.js`
5. Add your images to the `images` directory

## Deployment

This website can be deployed using GitHub Pages:

1. Go to your repository settings
2. Navigate to the "Pages" section
3. Select your main branch as the source
4. Your site will be published at `https://[your-username].github.io/[repository-name]`
